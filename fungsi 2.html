Jenis-Jenis Aplikasi Berbasis Web

Web Media Sosial

Platform yang memfasilitasi interaksi sosial antar pengguna seperti Facebook, Instagram, dan Twitter.

Web Berbasis Sistem Informasi

Software untuk mengelola data organisasi seperti sistem informasi akademik dan akuntansi.

Web E-Commerce

Platform jual beli online seperti Tokopedia, Shopee, Lazada, Amazon.

Web Pencarian

Mesin pencari seperti Google yang membantu menemukan informasi.

Web Informasi dan Berita

Platform seperti CNN, Detik.com, dan Kompas.com.

Web Server

Software seperti Apache, Nginx, IIS yang memproses permintaan client.

Web Browser

Software untuk membuka website seperti Chrome, Firefox, dan Edge.
